# Datei_M

**Datei_M** ist ein Desktop-Tool zum Sortieren, Analysieren und Organisieren von Dateien und Dokumenten.

## Beschreibung

Datei_M bietet eine umfassende Lösung zur Verwaltung und Organisation Ihrer Dateien und Dokumente. Das Tool verfügt über vier leistungsstarke Module, die Ihnen helfen, große Ordnerstrukturen zu bewältigen, wichtige Dokumente zu identifizieren und Texte effizient zu klassifizieren.

## Module

### 1. Dateien Manager
Automatisches Aufräumen und Organisieren von Ordnern nach konfigurierbaren Regeln.
- Sortierung nach Dateityp, Datum oder benutzerdefinierten Kriterien
- Automatische Ordnerstrukturierung
- Duplikaterkennung

### 2. Dokumente Manager
Intelligente Dokumentenverwaltung mit Regelengine und OCR-Unterstützung.
- Regelbasierte Dokumentenverarbeitung
- OCR-Integration zur Texterkennung in Bildern und PDFs
- Automatische Kategorisierung von Rechnungen, Briefen, etc.

### 3. Regex Editor
Komfortabler Editor zum Erstellen und Testen von Suchmustern.
- Visuelle Regex-Erstellung
- Live-Vorschau der Treffer
- Mustervorlagen für häufige Anwendungsfälle

### 4. Kategorie Manager
Trainieren und Verwalten von Kategorien für die Textklassifikation.
- Begriffe und Schlüsselwörter definieren
- Trainingsmodus für maschinelles Lernen
- Export/Import von Kategoriedefinitionen

## Installation

```bash
# Repository klonen
git clone https://github.com/ensergej-sketch/Datei_M.git
cd Datei_M

# Abhängigkeiten installieren
pip install -r requirements.txt

# Anwendung starten
python main.py
```

## Anforderungen

- Python 3.8 oder höher
- Weitere Abhängigkeiten siehe requirements.txt

## Verwendung

Starten Sie die Anwendung mit:
```bash
python main.py
```

Nach dem Start können Sie zwischen den vier Modulen wählen und mit der Organisation Ihrer Dateien beginnen.
